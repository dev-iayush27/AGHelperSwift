## Requirements

```ruby
- iOS 12 or Later
- Swift 5.x
- Xcode 11.x
```

## Installation

AGHelperSwift is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'AGHelperSwift'
```

Import it in the ViewController you want it to work:

```ruby
import AGHelperSwift
```
